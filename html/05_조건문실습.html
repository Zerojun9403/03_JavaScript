if (만약에) - else if(아니라면 이렇게, 저렇게) -else(모두아니면)

            앞에선 F 이면서 현재조건식 T

        }

    * 앞선 if / else if문 조건이 false인 경우
        이후 else if에서 해당 범위는 제외된다
